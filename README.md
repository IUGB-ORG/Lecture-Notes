## Computer Vision Series (Weeks 10–11)

The `CV_Lectures/` folder contains a standalone interactive HTML lecture series
covering Convolutional Neural Networks. All files are self-contained and open
directly in any modern browser — no server or installation required.

| File | Topic | Week |
|------|-------|------|
| `CV_Lecture1_Images_As_Data.html` | Images as Data | 10 |
| `CV_Lecture2_Convolution.html` | The Convolution Operation | 10 |
| `CV_Lecture3_CNN_Architecture.html` | CNN Architecture End-to-End | 10–11 |
| `CV_Lecture4_Historical_Victories.html` | Historical Victories (LeNet→ResNet) | 11 |
| `CV_Lecture5_Transfer_Learning.html` | Transfer Learning | 11 |
| `CV_Lecture_Filters.html` | CNN Filters & Kernels (deep dive) | supplement |
| `CNN_Face_Walkthrough.html` | How a CNN Sees a Face | supplement |
| `CNN_Explorer.html` | Pixel-Level Explorer — interactive tool | 10–11 |
| `CNN_Visual.html` | Visual Feature Map Explorer — interactive tool | 10–11 |

Access all CV materials via **`cv-lectures.html`** or directly from the
Week 10 and Week 11 lecture pages.
