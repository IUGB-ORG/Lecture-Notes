📐 About This Series

These interactive HTML lectures accompany Weeks 10–11 of CSC 4315 and form a complete self-contained introduction to Convolutional Neural Networks. The series progresses from first principles (how images are represented as data) through the convolution operation, full CNN architectures, the landmark models that defined the field, and modern transfer learning workflows. Deep-dive supplements and live interactive tools reinforce every concept.

Format: All files are standalone HTML — no internet connection or installation required. Open any file directly in a browser.

Transfer Learning
